OpenClaw HexonBot

HexonBot is an open-source hexapod robotics platform focused on mechanical clarity, modular electronics, and deeply hackable control systems.

GitHub Docs Hardware Community

Principles

Designed for experimentation. Everything is visible, adjustable, and meant to be taken apart — physically and conceptually.

Scope

Mechanics, electronics, firmware, and simulation — built as a coherent system, not a collection of parts.

Status

Active development. Expect iteration, refinement, and occasional breakage.
